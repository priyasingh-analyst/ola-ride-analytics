# 🚕 Ola Ride Analytics Dashboard

**Tool Used:** Power BI, DAX, Power Query  
**Type:** Data Analytics | Dashboard Design

##  Objective
Analyze Ola ride data to uncover peak hours, payment trends, 
and customer satisfaction patterns.

##  Key KPIs Tracked
- Total Rides & Revenue
- Peak Hours Analysis
- Payment Method Distribution
- Customer Ratings & Trends

##  Key Insights
- Identified peak ride hours for operational planning
- Analyzed most preferred payment methods
- Tracked customer rating trends over time

##  Skills Demonstrated
- Power BI Dashboard Design
- DAX Measures & Calculated Columns
- Data Cleaning & Transformation
- KPI Development
